# Reinforcement learning in tetris 

This is based on https://github.com/benycze/python-tetris implementation of tetris under the GPL-3.0 license
### Prerequisites

pygame

## Control


* *Arrows* - used for moving a block
* *Space*  - rotates the block
* *q*      - quit the game
* *p*      - pause the game
